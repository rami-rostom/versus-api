VERSUS - Documentation API

Les endpoints suivants permettent de récupérer et d'éditer les données de l'API.

Event

  • [GET] Get all events : /events
    ➔ https://versus-api.onrender.com/events
  • [GET] Get an event with id : /event/{id}
    ➔ https://versus-api.onrender.com/event/1
  • [GET] Get an event with slug : /event/{slug}
    ➔ https://versus-api.onrender.com/event/smash-bros-by-versus
  • [POST] Create a new event : /event
    ➔ https://versus-api.onrender.com/event
  • [DELETE] Delete an event : /event/{id}
    ➔ https://versus-api.onrender.com/event/1
  • [PATCH] Update an event : /event/{id}
    ➔ https://versus-api.onrender.com/event/1
  • [PATCH] Register at an event : /event/{id}/register
    ➔ https://versus-api.onrender.com/event/1/register
  • [PATCH] Unregister from an event : /event/{id}/unregister
    ➔ https://versus-api.onrender.com/event/1/unregister

User

  • [GET] Get all users : /users
    ➔ https://versus-api.onrender.com/users
  • [GET] Get an user : /user/{id}
    ➔ https://versus-api.onrender.com/user/1
  • [PATCH] Update an user : /user/{id}
    ➔ https://versus-api.onrender.com/user/1
  • [GET] Get all events of an user : /user/{id}/events
    ➔ https://versus-api.onrender.com/user/1/events
  • [GET] Get all teams of an user : /user/{id}/teams
    ➔ https://versus-api.onrender.com/user/1/teams
  • [PATCH] Update user prefered platforms : /user/{id}/preferences/platforms
    ➔ https://versus-api.onrender.com/user/1/preferences/platforms
  • [PATCH] Update user prefered games : /user/{id}/preferences/games
    ➔ https://versus-api.onrender.com/user/1/preferences/games

Authentification

  • [POST] Create a new user : /signup
    ➔ https://versus-api.onrender.com/signup
  • [POST] Sign-in : /login
    ➔ https://versus-api.onrender.com/login
